# 🩺 Medical Image Enhancement using Hybrid GAN-Transformer

This project presents a deep learning-based approach for enhancing medical images using a hybrid architecture that combines **Generative Adversarial Networks (GANs)** and **Transformer models**.

---

## 🚀 Features

- Upload medical images (X-ray, CT, MRI)
- AI-based enhancement using a hybrid model
- Real-time inference via Flask
- Clean and responsive user interface

---

## 🧠 Model

- Hybrid **GAN + Transformer** architecture  
- Trained on medical imaging datasets  
- Focused on improving **contrast** and **clarity**

---

## 🛠️ Tech Stack

- Python  
- Flask  
- PyTorch  
- HTML / CSS  

---
